源神: 现代化的异步版本管理工具
=========================

## 安装

```shell
cargo install yuan-shen
源神 启动!
```

## 使用

```shell
源神.exe

指令:
  启动!     创建一个新的源神世界
  观测      观测目标世界线与当前世界的差异
    --diffenence
    --diff
  异变
    --changes
  衍化
    --commit
  塌缩      将观测结果合并到当前世界线
    --squash
  收束      设定世界线收束节点
    --merge
  干涉      干涉目标世界线
    --rebase
  回溯      回溯到任意收束节点
    --reset
  退相干     从某一条世界线开始开启一个新世界
    --orphan
  跃迁      切换到指定名称的世界线
   --checkout
   -c
  branch
  stash
  逆城市化    这些对象有点太城市化了
    --garbage-collect
    --gc
  help    Print this message or the help of the given subcommand(s)

Options:
  -h, --help  Print help
```
