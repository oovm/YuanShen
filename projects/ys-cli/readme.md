```shell
Usage: 源神.exe <COMMAND>

指令:
  启动!       创建一个新的源神世界
  观测       观测目标世界线与当前世界的差异
  changes
  commit
  squash           将观测结果合并到当前世界线
  merge            设定世界线收束节点
  rebase           干涉目标世界线
  reverse          回溯到任意固化节点
  orphan           从某一条世界线开始开启一个新世界
  checkout         切换到指定名称的世界线
  branch
  stash
  garbage-collect  这些对象疑似有点太城市化了
  help             Print this message or the help of the given subcommand(s)

Options:
  -h, --help  Print help

```
